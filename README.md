# Akruti Plywood Website

A modern, responsive and professional business website for Akruti Plywood, designed to showcase plywood products, company information, applications, customer trust, catalogue and contact details.

## Live Website

https://akruti-plywood-website.vercel.app/

## Project Overview

This website is built for a plywood business to create a strong online presence and help customers easily explore products, understand the company, request quotations and contact the business directly.

## Features

- Modern responsive design
- Mobile-friendly layout
- Hero section with call-to-action
- About company section
- Product showcase
- Product advantages
- Applications section
- Gallery section
- Manufacturing process
- Testimonials
- Dealers section
- FAQ section
- Contact form
- WhatsApp contact button
- Catalogue section
- SEO-friendly meta tags

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- Lucide React
- EmailJS
- Vercel
